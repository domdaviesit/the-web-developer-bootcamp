Forms

action is where the form sends data to.

method is the type of HTTP method (get/post)

get is like a google search

post is where you post data to a data base

name attribute - you can give a name to the data you are collecting/sending. eg username = johndoe.

Labels add captions to the elements of the form.

Labels are great for ACCESSIBILITY

label for corresponds to input id (good for screen readers)

Validation adds rules to a form.eg password must be min 8 characters.

Some validation can be done in plain html. You can add required attribute to the input.

Data type validation. Does an email look like an email? eg @example.com. type = email.
